Flexbox Grid System By John Wiseheart

With the intent of improving my understanding of flexbox, I set out to write my own grid system. It turned out much simpler than I expected, and I will be using my own grid in future - its so simple.

The grid system supports both equal and unequal ratios of items, as well as breakpoints.

Equal Ratio Grid

The equal ratio grid works with simple row and column classes. Each column in a row is equally spaced.

Similarly, the unequal ratio grid works with row and column classes, as well as column-* classes. Depending on the number in the column-* class, the item will be at an n:1 ratio in width with a standard item.

Breakpoints

The grid system has 4 different breakpoints, set with an additional class row-*. Depending on how you set the breakpoint, the items will collapse into a single column with a specific range of screen sizes. Each of these rows have different breakpoints - try changing the width of your browser window.
